# ToDoList MVC em PHP + SQLite

## Requisitos

- PHP 8.1+ com PDO e SQLite3 habilitados

## Como Rodar

1. Clone o repositório:
```bash
git clone https://github.com/jpeodr/to-do-list.git
cd todolist_mvc
```

2. Inicie o servidor PHP:
```bash
php -S localhost:8000
```

3. Abra no navegador:
```
http://localhost:8000
```

## Banco de Dados

- O banco SQLite fica em `database/todolist.db`  
- Se não existir, será criado automaticamente ao rodar a aplicação.
